Python Revision – Clickable Mind Map

  • ▶ Python
    • ▶ Objects
      • Everything in Python is an object
      • Has identity, type, value
      • Variables reference objects
    • ▶ Classes
      • Blueprint for objects
      • Defines attributes and methods
      • Objects are instances of classes
    • ▶ Attributes
      • Names associated with an object
      • Data attributes → state
      • Method attributes → behavior
    • ▶ Mutability
      • Immutable: int, str, tuple
      • Mutable: list, dict, set
      • Design choice at class level
